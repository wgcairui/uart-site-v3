落地建议 (按优先级)

  1. 先装 tailwind.config.ts + ConfigProvider theme (1 个 PR 改 2 文件)
  2. 替换字体: Outfit + JetBrains Mono + Noto Sans SC (1 个 PR)
  3. admin 仪表盘 / 设备详情改 Bento 布局 (用现有 antd Table / Card 套 bento className, 不动 antd 内部)
  4. 登录页改 Glass + Mesh 渐变背景 (新文件, 独立 PR)
  5. 移动端沿用 Bento 2 列缩放
  6. 动效: hover 抬升 + status badge pulse + number 滚动
